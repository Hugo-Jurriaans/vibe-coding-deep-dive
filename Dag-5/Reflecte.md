- Kan je helpen leren bij het snappen van nieuwe code.

- Programmeren met AI bespaart veel tijd.

Risico's van AI voor developers
- Code is niet altijd optimaal en kan bugs bevatten.
- Bij veel gebruik kan je afhankelijk worden van AI.
- AI maakt programmeren minder leuk voor sommige developers.
- Mogelijk schrijft AI code anders dan hoe de devloper is gewend.
